# Copy Title and URL

This project is a firefox extension that adds a button in the address bar to copy the title and URL of the current tab.

## Installation

This web extension is unsigned, so you cannot install it the stable version of Firefox. See below.

> Unsigned extensions can be installed in the Developer Edition, Nightly, and ESR versions of Firefox,
after toggling the xpinstall.signatures.required preference in about:config.
https://extensionworkshop.com/documentation/publish/signing-and-distribution-overview/

1. Download `copy-title-and-url-<version>.xpi` from [releases][releases-latest] and install it

[releases-latest]: https://github.com/hexium310/copy-title-and-url/releases/latest
